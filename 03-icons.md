# Icons

## Icon style

Single-weight line drawings. Clean outlines, consistent stroke width, rounded rather than sharp terminals. Friendly and hand-drawn-feeling without being sketchy, which matches the small-batch, made-by-hand character. Keep line weight consistent across any set used together, give each icon even padding so it sits as part of a family, and pair icons with Kumbh Sans rather than a second typeface.

Note the tension between the stated style and the ethos artwork below, flagged in `OPEN-QUESTIONS.md`: the general rule says outline only, no fill and no shading, while the ethos icons carry selective solid fills and colour. Treat the ethos set as a deliberate exception, and keep any other icons (navigation, UI, editorial) to the pure outline rule.

## The ethos icons

The brand's most important icon set. They appear on the label and carry real meaning, so customers learn to read them as a consistent signal of what the brand stands for. Each is monochrome in one palette colour, with tints of that colour distinguishing fill from outline. The motifs are botanical and organic.

## The skin-type icon

An organic shape, the same silhouette for all five skin types, filled in that type's fixed hue. Inside sits the skin-type name in white, Roboto Condensed. Behind the front panel, the skin-type number (1 to 5) runs as a large Oyster `#E0E1E2` watermark. This is the single strongest wayfinding cue on the label and the site, so its colour is never borrowed for anything else.
