# Open questions

Things this system has surfaced that are not yet settled. Recorded here rather than quietly decided, so the loose threads stay visible.

## Icons

- **Icon style versus ethos artwork.** The stated icon style is single-weight outline, no fill and no shading. The ethos icons carry selective solid fills and colour. This system treats the ethos set as a deliberate exception and holds other icons to the outline rule, but the guidelines could say so explicitly.

## Wording and source drift

- **"Near Welcombe" in older docs.** Some source documents still read "near Welcombe" and need correcting to "in Welcombe".
- **Ethos icon captions and fragrance.** Confirm no ethos caption implies fragrance-free or essential-oil-free. The correct claim is "no synthetic fragrance".
